**How it works:**
`Scrapy Crawl` ‚Üí `Webhook / n8n` ‚Üí `Web Tech Audit (Lighthouse)` ‚Üí `AI Agent (Gap Analysis)` ‚Üí `DataForSEO (Keyword Research)` ‚Üí `Content Generation (TOFU/MOFU/BOFU)` ‚Üí `Social Publishing`

**Key Capabilities:**
- Autonomous website crawling and structured data extraction into Airtable
- Technical SEO audit via Google Lighthouse with AI-generated fixes (titles, alt text, content briefs via Gemini)
- Brand-aware content engine ‚Äî reads your own site before writing a single word
- Full multimedia publishing: Flux 1.1 for images, Kling 1.6 for video, auto-posted across 5 platforms
- Keyword research ‚Üí metadata ‚Üí full page ‚Üí internal links ‚Üí CMS publish via REST API, fully automated
